A modern, responsive website built for **Noma Tribe**, designed to showcase artists, music, merchandise, and upcoming events through an immersive and visually balanced user experience.

## Overview

Noma Tribe is a frontend web project focused on responsive design, clean layouts, and reusable components. The goal was to create a modern interface that maintains visual consistency across desktop and mobile while emphasizing readability, spacing, and hierarchy.

## Features

* Responsive design across desktop, tablet, and mobile
* Hero section with clear call-to-action
* Featured DJs showcase
* About section
* Merchandise section
* Upcoming events section
* Reusable UI patterns and consistent component styling
* Modern, maintainable CSS architecture

## Getting Started

Clone the repository:

```bash
git clone https://github.com/douriem/noma-website.git
```

Then open `index.html` in your browser or run the project using Live Server in Visual Studio Code.
